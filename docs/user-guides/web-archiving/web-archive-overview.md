# Overview

> **Purpose:** Web archives are organised into **public** and **restricted** collections:

**Collection Types:**
- **Public collections** - Contain web pages that are fully accessible to the general public
- **Restricted collections** - Include web pages captured with a "logged-in" view, which may contain premium or staff-only content

## Public Web Archive Collections

Public web archive collections can be accessed via:

- **[Preservica](https://icaew.access.preservica.com/index.php/SO_0c90c41b-e651-4c56-a503-b4088bab2f5a/)** – Takes you to the "Web Archives" folder in Preservica, which contains both public and restricted collections. **Ensure you are logged in** to access restricted collections.
- **[Archive-It](https://archive-it.org/organizations/1613)** – Directs you to the collections page for public web archives.

## Restricted Web Archive Collections

Restricted web archive collections can be accessed via (except for 2020 captures listed below):

- **[Preservica](https://icaew.access.preservica.com/index.php/SO_0c90c41b-e651-4c56-a503-b4088bab2f5a/)** – Access both public and restricted web archive collections. **Log in to Preservica** to view restricted collections.
