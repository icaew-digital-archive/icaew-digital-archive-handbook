# Overview

> **Purpose:** Preservica houses both **public** and **restricted** collections.

**Collection Types:**
- **Public collections** - Contain assets that are fully accessible to the public
- **Restricted collections** - Contain assets that require logging in to view

> **Access:** To access the restricted collections, please ensure you are logged in at the access portal: [https://icaew.access.preservica.com/](https://icaew.access.preservica.com/).

> **Note:** A username and password can be obtained upon request from the Digital Archive Manager or Digital Archivist.
